# Programação Modular

Repositório criado para armazenar exercícios e projetos desenvolvidos na disciplina de Programação Modular da faculdade.

## Tecnologias utilizadas
- Java
- Apache NetBeans
- IntelliJ IDEA

## Estrutura

- exercicios → programas feitos durante as aulas
- projetos → trabalhos maiores da disciplina
- anotacoes → resumos e conceitos importantes
